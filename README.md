# az-time-plan-picked
#### 周模板24小时 时间选择器
#### 周一 至 周日 七日每日24小时 多段 时段选择器
#### 拖拽添加时段、已添加时段点击左右两边拖拽修改时间、已添加时段点击主体拖拽修改时间、单日时段复制到其他日功能
[demo+doc 预览](https://mrchengg.github.io/az-time-plan-picked/demo)
